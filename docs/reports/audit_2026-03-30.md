## 🟡 Warnings (Nên sửa)

### 1. Monolith File — `App.jsx` (928 dòng, 22 functions)
- **Vấn đề:** Toàn bộ app nằm trong 1 file duy nhất. Khi thêm tính năng mới, file sẽ ngày càng khó maintain.
- **Ảnh hưởng:** Khó debug, khó review code, khó phân công nếu có nhiều người.
- **Cách sửa:** Tách thành component files:
  ```

  src/components/ObjectiveFilterView.jsx
  src/components/GlossaryView.jsx
  src/components/ResourcesView.jsx
  ```

- **Mức độ:** Không urgent — dự án hiện chỉ 1 developer, nhưng nên tách trước khi vượt 1500 dòng.

### 2. Empty Alt Tags — 34 `<img>` tags
- **Vấn đề:** 34 icon images dùng `alt=""` thay vì mô tả hình ảnh.
- **Ảnh hưởng:** Screen readers không đọc được → kém accessible. Tuy nhiên đây là decorative icons nên `alt=""` là intentional theo WCAG.
- **Verdict:** ✅ Acceptable — decorative icons nên để `alt=""` là đúng chuẩn. Không cần fix.

### 3. Package Version `0.0.0`
- **Vấn đề:** `package.json` version vẫn là `0.0.0` dù đã ở v2.3.
- **Cách sửa:** Đổi thành `"version": "2.3.0"` để align với git tags.

## 🟢 Suggestions (Tùy chọn)

### 1. Favicon chưa custom
- Đang dùng `/vite.svg` mặc định → nên thay bằng logo P2P Digital hoặc MediaGuide icon.

### 2. SPA Routing Fallback cho Production
- **Vấn đề:** React Router dùng `BrowserRouter` — khi deploy lên static hosting (Vercel/Netlify), cần cấu hình redirect tất cả routes về `index.html`.
- **Cách sửa:** Thêm `vercel.json` hoặc `_redirects` (Netlify):
  ```json
  { "rewrites": [{ "source": "/(.*)", "destination": "/index.html" }] }
  ```

### 3. ESLint outdated
- `eslint` v9.39.4 → v10.1.0 available. Minor update, không urgent.

### 4. CSS Monolith (`App.css` — 2079 dòng)
- Tương tự App.jsx, có thể tách CSS theo component khi project mở rộng.
- Hiện tại chưa gây vấn đề vì browser cache toàn bộ 1 file.
